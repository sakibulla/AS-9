GreenNest Functionality

User Authentication

Users can register with email and password.

Users can log in to access protected features.

Forgot password functionality with email reset.

Conditional rendering:

Logged-in users see their profile/avatar and a logout option.

Logged-out users see Login and Register buttons.

Browse Plants

Users can view a list of plants on the home page.

Each plant shows:

Image

Name

Short description

Price

Rating

Stock availability

Plant Details Page

Clicking a plant opens a detailed view with:

Large image

Full description

Price, rating, stock

“Book Consultation” Form:

Fields: Name, Email

Submission shows a success notification and clears the form

Responsive UI

Works well on desktop and mobile

Clean design using Tailwind CSS and DaisyUI

Notifications

Uses React Toastify to show success or error messages for:

Login/Register actions

Form submissions

Authentication events

Navigation

Navbar adapts based on authentication state

Links to:

Home

All Plants

Login/Register or Profile/Logout
